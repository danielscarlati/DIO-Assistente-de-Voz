# DIO-Assistente de Voz
Projeto do desafio Conversando por Voz com IA e Python

Nesse projeto eu criei um assistente de voz usando o Google Colab e ChatGPT

Com uma pitada de JavaScript, a gente gravou o audio que e esse arquivo: request_audio.wav e com ele criamos um speech 
to text usando Whisper. A gente integrou o ChatGPT para fazer o text to speech usando a biblioteca do Google gTTS, criando
o response_audio.wav.

Obs: Quando integrar com o ChatGPT, deve usar uma API key, no site da OpenAI ele gera uma aleatória.

Os resultados do notebook que criei estão na pasta results, com dois audios e os textos que foram usados durante o 
laboratorio.
